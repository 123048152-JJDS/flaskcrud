# Flask CRUD — Dockerizado

Proyecto original por: [Javier Moya](https://github.com/javiermoya2504/flaskcrud)

Modificaciones realizadas:
- Dockerización con Dockerfile y docker-compose.yml
- Configuración de healthcheck para MySQL
- Corrección de rutas de templates y static files
- Deploy automático con Railway

## Cómo correr localmente
```bash
docker-compose up --build
```

## Tecnologías
- Python / Flask
- MySQL 8.0
- Docker / Docker Compose
